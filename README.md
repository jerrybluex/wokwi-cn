# wokwi-cn

> 在浏览器里学 Arduino 单片机的教学产品。这是工作目录，PRD 是当前的「源起文件」。

## 工作目录

```
wokwi-cn/
├── README.md         # 你在这里
├── PRD.md            # 产品需求文档 — 唯一源起文档
└── docs/             # 后续：用户访谈、决策记录、设计稿
```

## 项目状态

- ✅ 已完成：技术 PoC（`../wokwi-clone.html`），2100 行前端仿真 demo
- 🚧 进行中：PRD v1 起草 — 等你过目和修订
- ⏳ 待做：用户访谈、技术选型、UI/UX、原型
- ❓ 未定：项目名（暂用 `wokwi-cn`，需正式命名）

## 怎么用这个目录

1. 先读 [PRD.md](./PRD.md)，整体过一遍产品和目标
2. 在 PRD 里用评论（或者直接在文档下方）写下你不同意 / 想改的地方
3. 重要决策会落到 `docs/decisions.md`，保持可追溯
4. `docs/user-research.md` 用来记录和老师的访谈（等你做了我再帮你整理）

## 下一步

等你确认 PRD 主线没问题，下一步我们一起做：
- 起个产品名（2-3 个候选，看你的偏好）
- 找 2-3 个一线老师做 30 分钟访谈
- 选技术栈、列里程碑
