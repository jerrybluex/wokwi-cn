# wokwi MVP 内测脚本
# 受试者：高职学生（物联网/创客/选修方向）
# 任务：完成 "LED 闪烁" 课程 5 步
# 时长：建议 30 分钟以内

## 前提条件
- 学生已登录 wokwi 账号
- 浏览器：Chrome / Firefox / Safari 最新版
- 网络：可访问 http://localhost:5173（或正式环境 URL）

> **2026-07-02 产品经理订正**:
> - 第 1 部分:只支持邮箱注册(PRD §6.7),无手机号
> - 第 4 部分:R 旋转未实现(CanvasPanel 只支持 Delete / Esc),改测拖拽到新位置
> - 第 6 部分:DEEPSEEK_API_KEY 不到位时 AI 部分跳过,记 fallback 路径
